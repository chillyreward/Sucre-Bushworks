# Sucre Bushworks Image Prompts

If local image files do not exist yet, you can generate them using an AI image generator (like Midjourney, DALL-E 3, etc.) using the exact prompts below.

## General Style Requirements
All images must adhere to the following rules:
- **No People**: Do not include human figures, faces, or crowds.
- **Premium**: Cinematic lighting, realistic photography, sharp focus.
- **Color Palette**: Forest green, warm sand, natural earthy tones.
- **Vibe**: Clean luxury outdoor brand, safari-modernist, Kenyan wilderness.
